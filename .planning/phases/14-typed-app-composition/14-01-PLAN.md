# Phase 14, Plan 1: Typed App Composition

**Status:** COMPLETE (implemented in session 2026-03-01, branch
worktree-typed-composition)

## Objective

Remove all `as any` / forced casts from the @fresh/effect hydration and plugin
layer. Store per-request state (hydration maps + Effect runner) in module-level
WeakMaps keyed on `ctx`, not on `ctx.state`. Make hydration functions generic
over the host state type `S`. Expose `runEffect(ctx, eff)` as an honest
`Promise<A>` return for plugin handlers.

## Requirements Covered

- COMP-01: Generic `setAtom<A,S>`, `serializeAtomHydration<S>`,
  `initAtomHydrationMap<S>`
- COMP-02: `runEffect(ctx, eff): Promise<A>` — no Effect-as-Response cast
- COMP-03: WeakMap-based per-request state
- COMP-04: `createCounterPlugin<S = unknown>(): App<S>`

## Changes Made

### `packages/effect/src/hydration.ts` — complete rewrite

- Module-level WeakMaps replace `ctx.state[Symbol]` pattern:
  ```ts
  const hydrationMaps = new WeakMap<object, Map<string, unknown>>();
  const requestRunners = new WeakMap<object, RequestRunner>();
  ```
- `SerializableAtom<A>` typed interface replaces `(atom as any)`:
  ```ts
  interface SerializableAtom<A> {
    readonly [Atom.SerializableTypeId]: {
      key: string;
      encode: (value: A) => unknown;
    };
  }
  ```
- All three exported functions generic over `S = unknown`:
  ```ts
  export function setAtom<A, S = unknown>(ctx: { state: S }, atom, value): void;
  export function serializeAtomHydration<S = unknown>(
    ctx: { state: S },
  ): string | null;
  export function initAtomHydrationMap<S = unknown>(ctx: { state: S }): void;
  ```
- New internal + public API:
  ```ts
  export function _setRequestRunner(ctx: object, runner: RequestRunner): void;
  export function runEffect<A, R = never>(
    ctx: object,
    eff: Effect<A, unknown, R>,
  ): Promise<A>;
  ```

### `packages/effect/src/app.ts`

- Added `_setRequestRunner` middleware in `createEffectApp()`:
  ```ts
  app.use((ctx) => {
    _setRequestRunner(ctx, (eff) => resolver(eff, ctx) as Promise<unknown>);
    return ctx.next();
  });
  ```

### `packages/effect/src/mod.ts`

- Added `runEffect` to public exports

### `packages/examples/typed-composition/counter_plugin.tsx`

- `effectRoute` removed; replaced with `runEffect`
- Factory made generic: `createCounterPlugin<S = unknown>(): App<S>`
- `setAtom(ctx, ...)` calls need no cast — `ctx: Context<S>` satisfies
  `{ state: S }`
